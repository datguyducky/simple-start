Simple Start is minimalistic extension for Chrome and Firefox that lets you have all the bookmarks you need every time you open new tab.

Bookmarks tags

Organize bookmarks in tags so you can have every bookmark you may need in specific situation. Be more productive thanks to seperated tags for your home, work, and your hobbies.

Save any website

Unlike "Firefox Home Content", Simple Start lets you to save every website you like to.

Synchronize

Thanks to Firefox/Google Chrome Sync you are available to synchronize bookmarks and tags used by Simple Start between your devices.

Dark Mode

You work at night? Or maybe you don't like to see this bright white every time you opens new tab? No problem, Simple Start comes with built-in darkmode.

Minimalistic

This extension was built with minimalistic style in mind. It comes only with functions that you may need and looks simple rather than being overpacked with everything.

Customize

You want to change background color? No problem! You don't like how big or small gap between bookmarks is? No problem, you can change that and more via built-in settings menu.
(more options will come in future updates!)
