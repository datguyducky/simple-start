Frequently Asked Questions

Getting started:

  1. Simple Start is an extension for Firefox which replaces your new tab and home page with customizable, minimalistic list of bookmarks that you may need every time you open new tab.

  2. For Firefox: Visit Simple Start page on Firefox Add-ons and click on 'Add to Firefox' button.

Bookmarks and tags:

  1. You can use build-in option in Firefox to add new bookmark (remember to save it to 'simplestart' folder) or you can click on '+' button while being on new tab page (be sure that page you want to bookmark have 'https://' or 'http://' at the beginning.)

  2. Unfortunately currently Simple Start don't comes with option to delete or edit bookmarks but you can use Firefox built-in Bookmark Library to delete or edit bookmark of your choice.

  3. Simple Start lets you organize your bookmarks into different categories (e.g. to separate your favorite websites related to work or your hobies). Is really simple to create tag - just click on '+' button while being on new tab page with Simple Start installed.

  4. Unfortunately currently Simple Start don't comes with option to easy rearrange your bookmarks but you can use Firefox built-in Bookmark Library to move bookmarks higher or lower so they can be different arranged on new tab page.

Synchronization:

  1. Simple Start uses Firefox build-in option to synchronize bookmarks between devices. You need to login into Firefox account (and select appropriate option on Sync Settings page) to let Firefox sync your bookmarks between devices.

  2. Synchronization in Simple Start lets you use all bookmarks and tags you've saved between your devices without need to add them again.

  3. Is not required to turn-on Sync on Firefox to use Simple Start extension but I recommend this option if you want to easily use all your bookmarks and tags between devices.

Customization:

  1. Currently Simple Start lets you customize: background color, gap between cards, size of icons for bookmarks + Dark Mode is available.

  2. Yes! I plan to add as much as possible options, so users of this extension can customize almost every aspect of their new tab page with Simple Start installed.

Troubleshooting:

  1. Check zoom settings of your new tag page. Zoom-in our zoom-out the page.

  2. Check if you login correctly into your Firefox account or you can try to force sync. Have in mind that sometimes sync takes some time to completely sync everything.

  3. You can delete Simple Start extension from 'Manage Your Extensions' page in Firefox by clicking three dots next to name of an extension and then by clicking on 'delete' button.
